## Part 1: How to Check Domain Availability

### Method 1: Namecheap (Recommended)
```
1. Go to: https://www.namecheap.com/domains/
2. Search: "velora" (or your choice)
3. Look for .com availability
4. Check price (should be $8-15/year)
```

### Method 2: GoDaddy
```
1. Go to: https://www.godaddy.com/domains
2. Search domain
3. Filter: Show only .com
```

### Method 3: Whois Lookup
```
1. Go to: https://whois.domaintools.com/
2. Enter: velora.com
3. Check status: "Available" or "Registered"
```

## Part 2: Trademark Checking

### USPTO Database (US)
```
1. Go to: https://www.uspto.gov/trademarks/search
2. Search: "velora" in "Basic Word Mark Search"
3. Filter by: Class 36 (Financial) and Class 42 (Software)
4. Check: Any live registrations?
```

### EUIPO (Europe)
```
1. Go to: https://euipo.europa.eu/eSearch/
2. Search trademark
3. Check Nice Classes: 36, 42
```

### Google Test
```
1. Google: "Velora fintech" or "Velora trading"
2. Check: Any direct competitors?
3. Look for: Company websites, not generic results
```

## Part 8: Red Flags to Avoid

### Don't Choose a Name If:

1. ‚ùå **Domain is .io/.ai only** (unless you're okay with it)
   - .com adds credibility for institutional clients
   - .io/.ai = tech startup vibe (less serious)

2. ‚ùå **Trademark conflict exists**
   - Even if domain is available
   - Check USPTO before committing

3. ‚ùå **Hard to spell after hearing**
   - "Is that V-E-L-O-R-A or V-E-L-L-O-R-A?"
   - Test with 3 friends: Say it once, ask them to write it

4. ‚ùå **Negative associations**
   - Google the name + see what comes up
   - Avoid names with bad connotations

5. ‚ùå **Too similar to competitor**
   - Don't pick "QuantFlow" if QuantConnect exists
   - Legal risk + brand confusion
